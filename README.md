# M-S-WIFI
WIFI IDS

git clone https://github.com/murtadhakareemm/M-S-WIFI.git

cd M-S-WIFI

sudo python3 M-S.py




المتطلبات الأساسية: يجب أن يتوفر لديك كرت شبكة لاسلكي (Wi-Fi Adapter) يدعم وضع المراقبة (Monitor Mode) وحقن الحزم (Packet Injection)، مثل كروت Alfa Network أو TP-Link.

الأنظمة الوهمية (Virtual Machines): إذا كنت تستخدم كالي لينكس كنظام وهمي (VMware / VirtualBox)، لا يمكنك استخدام كرت الواي فاي المدمج في اللابتوب. ستحتاج إلى توصيل كرت شبكة خارجي (USB Wi-Fi Adapter) وربطه مباشرة بالنظام الوهمي.

التثبيت التلقائي: عند التشغيل للمرة الأولى، سيقوم النظام تلقائياً بفحص وتثبيت المكتبات المطلوبة (scapy, customtkinter, arabic-reshaper, python-bidi). يرجى التأكد من اتصالك بالإنترنت في أول تشغيل فقط.
